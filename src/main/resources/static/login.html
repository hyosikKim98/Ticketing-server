Login

API 로그인

`POST /api/auth/login`에 맞춰 이메일과 비밀번호를 전송합니다.

아직 로그인하지 않았습니다.

Flow

현재 화면에서 가능한 동작

1. 로그인 후 access token을 브라우저 localStorage에 저장
2. 이벤트 구매 페이지에서 이벤트 조회
3. 대기열 진입 후 대기 페이지에서 순번 조회
4. 입장 토큰 발급은 관리자 `issue` API가 별도로 필요
